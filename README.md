Benevolent Bridges – PCL Project

Benevolent Bridges is a web project developed using HTML and CSS to support and create awareness for beggars, underprivileged individuals, homeless people, and those in need of help.
The project aims to bridge the gap between generosity and necessity by connecting donors with people who require support.

Project Structure:
index.html     → Main webpage
style.css      → Stylesheet for styling
images/        → (optional) supporting images
